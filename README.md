Step by Step DNA sequence analysis web application

A minimized tool (web application) for DNA sequence analysis. This single web page workflow is for the beginners of biological bachelor students.

---

Features

- Upload and convert ab1 format file to FASTA format
- Show DNA sequence and its chromatograph
- Generate its protein sequences with 3 reading frames
- Provide manual correction of sequence errors
- Identify and highlight sequence domains
- Annotation and export the results
